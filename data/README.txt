Dataset Information

The original dataset used in this project is too large to be stored directly in this GitHub repository.

You can access the full dataset using the Google Drive link provided in the main README file of this repository.

Dataset Details:
- Name: U.S. Crime Dataset
- Source: Kaggle
- Records: 638,454
- Time Period: 1980–2014

Note:
Due to GitHub file size limitations, the dataset is hosted externally.
